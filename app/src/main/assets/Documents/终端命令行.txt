终端命令行
1.pwd：(print working directory的缩写)显示当前文件夹的绝对路径
2.ls：（List directory contents）列出目录下的文件和文件夹
3.cd：（change directory）
    cd 无参数，进入初始默认根目录;
    cd ~ 返回默认根目录；
    cd folder 进入此目录；
    cd .. 进入父级目录;
    cd - 返回上一个访问的目录，可以用于两个目录的来回切换。
4.rm 文件：（remove）彻底删除文件
5.rmdir：（remove directory）彻底删除文件夹
6.cp 文件名 目标目录：（copy）复制文件
7.mv：（move）移动文件
8.mkdir：（make directory）创建文件夹

9.cat：文件名(|less)　　在终端下查看文件
~代表root　　如：~/Document/CPP2/
mkdiv　　　　　　新建文件夹
g++ 源文件名　　　　编译源文件，产生a.out
./文件名　　　　　　运行  例如：./a.out < 输入文件名 > 输出文件名
control+d　　　　　中断a.out运行
nano 　　　　　　编写脚本语言　　ctrl+o存储
nano ....sh　　　　打开
bash ....sh　　　　运行脚本
echo "...$i..."　　　输出语句
tar -zxf abc.tar.gz     tar文件解压
ssh root@192.168.1.222   以root账号远程连接222服务器
unrar x abc.rar     rar文件解压，需要安装rar工具
