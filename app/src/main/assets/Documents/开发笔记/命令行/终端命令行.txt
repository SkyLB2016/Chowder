终端命令行：主Mac，次Linux
1.小知识：
    tab键，自动补全目录名称。
    清屏：Ctrl + L
    中止执行：Ctrl + C
    ~代表根目录
    以.开头的文件或文件夹，即为隐藏文件
    .代表当前目录，..代表上级目录，多一个点多上一级。
    &表示同时执行命令，&&表示串行执行命令。
    光标至行首行尾：Ctrl + A与Ctrl + E
    删除至行首行尾：Ctrl + U与Ctrl + K
    删除至单词开始：Ctrl + W
    搜索历史命令：Ctrl + R
    Mac上选中即复制，单击滚轮粘贴。
    按住 ⌘，单击路径可以在 Finder 中打开（必须是绝对路径）
2.快捷小命令
    pwd：(print working directory的缩写)显示当前文件夹的绝对路径。
    man 具体命令：查阅此命令的说明
    say "hello" 朗读hello
    open . ：打开当前目录
    history （num）：显示使用过的历史命令
    file （-b不显示名称）文件：显示文件信息
    which 名：显示$path里配置的信息
    who 显示系统中有哪些使用者
    echo 输出
    sleep 10 睡10秒
3.ls：（List directory contents）列出目录下的文件和文件夹
    -l：列出详细的目录清单
    -a：连隐藏文件一起显示
    -la（al）：详细清单，包含隐藏文件的
4.cd：（change directory）
    cd 无参数：进入初始默认根目录;
    cd ~ 返回默认根目录；
    cd folder：进入此目录；
    cd .. ：进入父级目录
    cd ../.. ：进入父级的父级目录;==cd ...
    cd - ：返回上一个访问的目录，可以用于两个目录的来回切换。
5.cp 文件名 目标目录：（copy）复制文件
    -a 文件路径 复制文件路径：连同文件的权限属性一起复制
    -r 文件路径 复制文件路径：复制文件夹
    -ra 文件夹路径 复制文件夹路径：连同文件夹的权限属性一起复制
    -rav 文件夹路径 复制文件夹路径：连同文件夹的权限属性一起复制，加v，可以看到复制过程
6.rm 文件：（remove）彻底删除文件
    -f 文件：Linux需要yes确认，加f后不需要
    -r 文件夹：删除文件夹；
    -r *：删除此目录下的内容
7.mv 文件 文件路径或名称：（move）移动文件，重命名文件
8.touoch a b c 创建文件，单个多个
    mkdir 目录1 目录2：（make directory）创建文件夹，单个多个
    mkdir -p 目录1/目录2：创建层级嵌套文件夹
9.cat （-n显示行号）文件名：在终端下查看文件，全部输出，ctrl+o 终止输出；
    less 文件名：在终端下查看文件，单页输出；
    more 文件名：与less一样；
    ctrl+f（space） 下页，ctrl+b 上页，不加ctrl也可以；
10.其他
    nano 　　　　　　编写脚本语言　　ctrl+o存储
    nano ....sh　　　　打开
    bash ....sh　　　　运行脚本
    echo "...$i..."　　　输出语句
    tar -zxf abc.tar.gz     tar文件解压
    ssh root@192.168.1.222   以root账号远程连接222服务器
    unrar x abc.rar     rar文件解压，需要安装rar工具

    curl 网址 ：查看网址源码
    curl -o 文件名 网址
